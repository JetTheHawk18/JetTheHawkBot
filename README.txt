This Source Code is for both of my Jet the Hawk bots

main.py is from my main bot that can play Jet the Hawk's theme song
mainmini.py is my demo version of my bot.

Packages required:
- discord.py
- ffmpeg
- PyNaCl
Also needed from you
- A Discord application with bot and your bot token and is added to your discord server.
- A hosting provider (or you can self host it if you want)
- Some knowledge of Discord.py

========================================

SETUP
1. Make sure you install the discord.py ffmpeg and PyNaCl packages.
2. Extract the images.zip archive to extract the folder with the images. Have the images folder in the root path of your bot files.
3. Replace "BOT TOKEN HERE" (at the bottom with your bot token
4. Run the Bot in your hosting provider
